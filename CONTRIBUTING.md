# Contributing

Thanks for helping improve `pbd-playground`.

## Local workflow

1. Install Node.js 22 or newer.
2. Run `make install-hooks`.
3. Run `make test` before pushing.
4. Use Conventional Commits, such as `feat: add rope tearing`.

## Guidelines

- Keep the public app static and GitHub Pages friendly.
- Do not commit secrets, generated local logs, or private configuration.
- Add focused tests for physics, interaction, and rendering glue when changing behavior.
