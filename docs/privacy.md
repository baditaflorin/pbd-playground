# Privacy

Live app: https://baditaflorin.github.io/pbd-playground/

`pbd-playground` collects no analytics and sends no telemetry.

The app stores small local preferences in the browser with `localStorage`:

- selected body preset
- selected tool
- stiffness
- wind
- audio enabled
- visual quality

No cookies, accounts, private keys, payment data, or simulation state are collected by the app.
